dolfin_mech.problems.problem_hyperelasticity

Defines the Hyperelasticity class.

Comprehensive manager for nonlinear finite strain simulations. It automates the setup of large-deformation kinematics, complex hyperelastic constitutive laws, and mixed formulations required for incompressible and nearly-incompressible material behavior.

Bases: dolfin_mech.problems.problem.Problem

A general problem class for finite strain hyperelasticity.

This class handles the setup and solution of nonlinear elasticity problems where the material behavior is described by a strain energy density function \(\Psi(\mathbf{C})\). It supports both compressible and incompressible formulations.

Formulation: The problem minimizes the total potential energy \(\Pi(\mathbf{u})\):

\[\Pi(\mathbf{u}) = \int_{\Omega_0} \Psi(\mathbf{F}) \, dV - W_{ext}\]

If w_incompressibility is True, a mixed \(\mathbf{u}-p\) formulation is used to enforce the constraint \(J = 1\) (or near-incompressibility) via a Lagrange multiplier \(p\):

\[\Pi(\mathbf{u}, p) = \int_{\Omega_0} \Psi_{dev}(\bar{\mathbf{F}}) \, dV + \int_{\Omega_0} p(J-1) \, dV - W_{ext}\]
Parameters:

  • w_incompressibility (bool) – If True, enables the mixed formulation adding a pressure sub-solution to handle volumetric locking.

  • mesh – The computational mesh.

  • displacement_degree – Polynomial degree for the displacement field.

  • elastic_behavior – Dictionary defining a single material model.

  • elastic_behaviors – List of dictionaries for multi-material setups.

add_pressure_subsol(name=None, degree=0)[source]

Adds the pressure scalar field sub-solution (Lagrange multiplier).

This is used only when w_incompressibility is True. To satisfy the LBB condition (inf-sup condition) and avoid volumetric locking, the pressure degree is typically lower than the displacement degree (e.g., P2-P1).

set_quadrature_degree(quadrature_degree=None)[source]

Sets the integration order for the variational forms.

Can be an integer, “full” (exact), or “default” (optimized). The “default” mode attempts to use a reduced quadrature rule that is computationally efficient while maintaining accuracy for standard element types.

set_kinematics(add_fois=True)[source]

Initializes the kinematic quantities derived from displacement.

Calculates:

  • F: Deformation Gradient (\(\mathbf{I} + \nabla \mathbf{u}\))

  • J: Jacobian determinant (\(\det \mathbf{F}\))

  • C: Right Cauchy-Green deformation tensor (\(\mathbf{F}^T \mathbf{F}\))

  • E: Green-Lagrange strain tensor (\(0.5 (\mathbf{C} - \mathbf{I})\))

Parameters:

add_fois – If True, registers these tensors as Fields of Interest for output.
