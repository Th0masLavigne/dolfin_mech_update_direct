dolfin_mech.operators.linearizedelasticity

Defines the LinearizedElasticity class.

Implements the internal virtual work for small-strain mechanics, relating the Cauchy stress tensor to the infinitesimal strain tensor within a linear elastic framework.

Module Contents

class dolfin_mech.operators.linearizedelasticity.LinearizedElasticity(kinematics, u_test, material_model, material_parameters, measure)

Bases: dolfin_mech.operators.operator.Operator

Operator representing the internal virtual work for linearized elasticity.

This class assembles the residual variational form for a body under the assumption of small deformations and small strains. It relates the Cauchy stress tensor \(\mathbf{\sigma}\) to the symmetric part of the displacement gradient (the infinitesimal strain tensor \(\mathbf{\epsilon}\)).

The internal virtual work \(\delta \Pi_{int}\) is defined as:

\[\delta \Pi_{int} = \int_{\Omega} \mathbf{\sigma} : \delta \mathbf{\epsilon} \, d\Omega\]
where:

  • \(\mathbf{\sigma}\) is the Cauchy stress tensor provided by the material model.

  • \(\delta \mathbf{\epsilon} = \text{sym}(\nabla \delta \mathbf{u})\) is the virtual infinitesimal strain tensor.

Attributes: kinematics (Kinematics): Kinematic variables associated with the small strain assumption. material (Material): Linear elastic material instance created via the factory. measure (dolfin.Measure): Integration measure (typically dx). res_form (UFL form): The resulting residual variational form.

Parameters:

  • kinematics (dmech.Kinematics) – Kinematics object (typically handling infinitesimal strains).

  • u_test (dolfin.TestFunction) – The test function (virtual displacement).

  • material_model (str) – Name of the material model (e.g., “Hooke”).

  • material_parameters (dict) – Dictionary of material properties (e.g., E, nu).

  • measure (dolfin.Measure) – Dolfin measure for domain integration.
