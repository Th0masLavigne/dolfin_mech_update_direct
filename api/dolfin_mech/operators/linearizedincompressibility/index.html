dolfin_mech.operators.linearizedincompressibility

Defines the LinearizedIncompressibility class.

Assembles the linearized volumetric constraint (divergence-free condition) to enforce material incompressibility within a small-strain mixed variational framework.

Module Contents

class dolfin_mech.operators.linearizedincompressibility.LinearizedIncompressibility(kinematics, p_test, measure)[source]

Bases: dolfin_mech.operators.operator.Operator

Operator enforcing the incompressibility constraint in a linearized (small strain) elasticity framework.

This operator is the counterpart to the LinearizedHydrostaticPressureOperator. In a mixed displacement-pressure formulation, it assembles the equation governing the Lagrange multiplier (pressure) to enforce that the material volume remains constant under small deformations.

The residual contribution is:

\[\mathcal{R}_{incomp} = - \int_{\Omega} \text{tr}(\mathbf{\epsilon}) \delta p \, d\Omega\]
where:

  • \(\mathbf{\epsilon}\) is the infinitesimal strain tensor.

  • \(\text{tr}(\mathbf{\epsilon}) = \nabla \cdot \mathbf{u}\) is the linearized volumetric strain.

  • \(\delta p\) is the test function associated with the pressure field.

The constraint \(\text{tr}(\mathbf{\epsilon}) = 0\) effectively ensures that the displacement field is solenoidal (divergence-free).

Variables:

  • kinematics (Kinematics) – Kinematic variables providing the infinitesimal strain.

  • measure (dolfin.Measure) – Integration measure (typically dx).

  • res_form (UFL form) – The resulting residual variational form.

Parameters:

  • kinematics (dmech.Kinematics) – Kinematics object handling infinitesimal strains.

  • p_test (dolfin.TestFunction) – Test function associated with the pressure field (Lagrange multiplier).

  • measure (dolfin.Measure) – Dolfin measure for domain integration.
