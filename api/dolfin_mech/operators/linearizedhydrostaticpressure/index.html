dolfin_mech.operators.linearizedhydrostaticpressure

Defines the LinearizedHydrostaticPressure class.

Implements the coupling term between the pressure field and volumetric strain within a small-strain mixed variational framework.

Module Contents

class dolfin_mech.operators.linearizedhydrostaticpressure.LinearizedHydrostaticPressure(kinematics, u_test, p, measure)

Bases: dolfin_mech.operators.operator.Operator

Operator representing the virtual work of hydrostatic pressure in a linearized (small strain) elasticity framework.

In mixed displacement-pressure formulations, this operator defines the coupling term between the pressure field \(p\) and the displacement test function \(\delta \mathbf{u}\). Physically, it represents the internal virtual work done by the pressure during a volumetric change.

The internal virtual work contribution \(\delta \Pi_{pres}\) is:

\[\delta \Pi_{pres} = - \int_{\Omega} p \, \text{tr}(\delta \mathbf{\epsilon}) \, d\Omega\]
where:

  • \(p\) is the hydrostatic pressure (often a Lagrange multiplier).

  • \(\delta \mathbf{\epsilon} = \text{sym}(\nabla \delta \mathbf{u})\) is the virtual infinitesimal strain tensor.

  • \(\text{tr}(\delta \mathbf{\epsilon}) = \nabla \cdot \delta \mathbf{u}\) represents the virtual linearized volumetric strain.

Variables:

  • kinematics (Kinematics) – Kinematic variables associated with the small strain problem.

  • p (dolfin.Function or dolfin.Constant) – The hydrostatic pressure field.

  • measure (dolfin.Measure) – Integration measure (typically dx).

  • res_form (UFL form) – The resulting residual variational form.

Parameters:

  • kinematics (dmech.Kinematics) – Kinematics object handling infinitesimal strains.

  • u_test (dolfin.TestFunction) – The displacement test function (virtual displacement).

  • p (dolfin.Function, dolfin.Constant, or dolfin.TrialFunction) – Hydrostatic pressure field or constant.

  • measure (dolfin.Measure) – Dolfin measure for domain integration.
