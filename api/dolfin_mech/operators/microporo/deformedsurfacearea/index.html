dolfin_mech.operators.microporo.deformedsurfacearea

Defines the DeformedSurfaceArea class.

Tracks the evolution of boundary surface area under large deformations using Nanson’s formula, enabling the simulation of area-dependent phenomena such as surface tension.

Module Contents

class dolfin_mech.operators.microporo.deformedsurfacearea.DeformedSurfaceArea(S_area, S_area_test, kinematics, N, measure)

Bases: dolfin_mech.operators.operator.Operator

Operator representing the evolution of surface area on a boundary under large deformations.

This operator facilitates the tracking of the current surface area \(S\) relative to its initial area \(S_0\). It is particularly useful for problems involving surface-dependent phenomena like surface tension or surfactant concentration in lung mechanics.

The change in area is governed by Nanson’s formula, which relates the reference normal \(\mathbf{N}\) and reference area element \(dS\) to the deformed normal \(\mathbf{n}\) and deformed area element \(ds\):

\[\mathbf{n} \, ds = J \mathbf{F}^{-T} \mathbf{N} \, dS\]

The scalar ratio of area change \(T \cdot J\) (where \(T = \|\mathbf{F}^{-T} \mathbf{N}\|\)) is enforced in weak form:

\[\mathcal{R} = \int_{\Gamma} \left( \frac{S}{S_0} - J \|\mathbf{F}^{-T} \mathbf{N}\| \right) \delta S \, d\Gamma\]
Variables:

  • measure (dolfin.Measure) – Boundary measure (typically ds).

  • kinematics (Kinematics) – Kinematic variables providing the deformation gradient \(\mathbf{F}\) and Jacobian \(J\).

  • N (dolfin.Constant) – The unit normal vector in the reference configuration.

  • res_form (UFL expression) – The resulting residual variational form.
