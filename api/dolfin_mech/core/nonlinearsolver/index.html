dolfin_mech.core.nonlinearsolver

Defines the NonlinearSolver class.

Provides an iterative Newton-Raphson framework for solving nonlinear problems with support for advanced linear backends (PETSc/MUMPS), adaptive relaxation techniques, and modal analysis via eigenvalue solving.

Module Contents

class dolfin_mech.core.nonlinearsolver.NonlinearSolver(problem, parameters, relax_type='constant', relax_parameters={}, print_out=True, write_iter=False)

Class for solving nonlinear systems of equations using iterative methods.

This solver implements a Newton-Raphson scheme with support for various linear backends (PETSc/MUMPS) and relaxation (line-search) techniques to improve convergence for highly nonlinear problems.

Parameters:

  • problem (dolfin_mech.problem.Problem) – The mechanical problem instance.

  • parameters (dict) – Solver parameters including ‘linear_solver_type’, ‘sol_tol’, and ‘n_iter_max’.

  • relax_type (str, optional) – Type of relaxation/line-search, defaults to “constant”.

  • relax_parameters (dict, optional) – Specific parameters for the chosen relaxation method.

  • print_out (bool or str, optional) – Output destination (‘stdout’, ‘argv’, or a filename), defaults to True.

  • write_iter (bool, optional) – If True, writes solution files for every Newton iteration, defaults to False.

Variables:

  • problem (Problem) – The nonlinear problem to be solved.

  • linear_solver_type (str) – The backend used for linear solves (‘petsc’ or ‘dolfin’).

  • relax_type (str) – The relaxation strategy (‘constant’, ‘aitken’, ‘gss’, ‘backtracking’).

  • sol_tol (list) – Convergence tolerances for each sub-solution.

  • n_iter_max (int) – Maximum number of Newton iterations allowed.

  • success (bool) – Whether the solver converged in the last solve call.

  • k_iter (int) – Current Newton iteration counter.

solve(k_step=None, k_t=None, dt=None, t=None)

Executes the nonlinear solve for a given time step.

Iteratively assembles the linear system, solves for the increment, updates the solution, and tests for convergence.

Parameters:

  • k_t – Current time step index.

  • dt – Time increment size.

  • t – Current total time.

assemble_linear_system()

Assembles the residual vector and Jacobian matrix.

This method handles standard integrals and special vertex-based integrals separately to accommodate specific dolfin constraints.

eigen_solve()

Solves the eigenproblem for the Jacobian matrix to identify modal shapes.

compute_dsol_norm()

Computes and prints the L2 norm of the solution increment.

compute_relax_constant()

Sets a constant relaxation factor \(\omega\).

compute_relax_aitken()

Computes the relaxation factor using the Aitken dynamic method.

compute_relax_gss()

Computes the optimal relaxation using a Golden Section Search on the potential energy.

compute_relax_backtracking()

Computes relaxation using a backtracking line-search until residual is finite.

update_sol()

Updates the solution vector by adding the relaxed increment.

compute_sol_norm()

Computes and prints L2 norms for the current and previous solutions.

compute_sol_err()

Computes the relative error between current and old solution norms.

exit_test()

Checks if the solution error is below the tolerance for all sub-solutions. Sets self.success accordingly.
