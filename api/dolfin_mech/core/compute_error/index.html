dolfin_mech.core.compute_error

Numerical utility functions for error analysis in finite element simulations.

Module Contents

dolfin_mech.core.compute_error.compute_error(val, ref)[source]

Compute the relative error or ratio between a value and a reference.

The error is calculated as:

\[\begin{split}e = \begin{cases} 0 & \text{if } \text{ref} \approx 0 \text{ and } \text{val} \approx 0 \\ 1 & \text{if } \text{ref} \approx 0 \text{ and } \text{val} \neq 0 \\ \frac{\text{val}}{\text{ref}} & \text{otherwise} \end{cases}\end{split}\]
Parameters:

  • val (float) – The current or calculated value to compare.

  • ref (float) – The reference or baseline value.

Returns:

The computed relative error or ratio.

Return type:

float

Notes

This function uses dolfin.near() with a tolerance of \(10^{-9}\) to handle floating point comparisons near zero.
