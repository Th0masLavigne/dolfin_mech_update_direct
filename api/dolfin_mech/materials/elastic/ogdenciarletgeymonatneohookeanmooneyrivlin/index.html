dolfin_mech.materials.elastic.ogdenciarletgeymonatneohookeanmooneyrivlin

Composite hyperelastic material combining Ciarlet-Geymonat, Neo-Hookean, and Mooney-Rivlin models.

This module provides a complex material implementation that independently parameterizes volumetric compressibility and shear response using the first and second invariants.

Module Contents

class dolfin_mech.materials.elastic.ogdenciarletgeymonatneohookeanmooneyrivlin.OgdenCiarletGeymonatNeoHookeanMooneyRivlin(kinematics, parameters, decoup=False)[source]

Bases: dolfin_mech.materials.elastic.elastic.ElasticMaterial

Class representing a composite hyperelastic material model.

Combines Ciarlet-Geymonat volumetric energy with a combined Neo-Hookean and Mooney-Rivlin deviatoric response.

This class serves as a high-level wrapper that allows for sophisticated material modeling where the volumetric behavior (compressibility) and the shear behavior (based on the first and second invariants) are independently controlled but additively combined.

The total strain energy density is:

\[\Psi_{total} = \Psi_{bulk}^{CG}(J) + \Psi_{dev}^{NH}(\mathbf{C}) + \Psi_{dev}^{MR}(\mathbf{C})\]
Where:

  • \(\Psi_{bulk}^{CG}\) is handled by OgdenCiarletGeymonatElasticMaterial.

  • \(\Psi_{dev}^{NH+MR}\) is handled by NeoHookeanMooneyRivlinElasticMaterial.

This class also provides auxiliary mechanical quantities such as the hydrostatic pressure and the Von Mises equivalent stress.

Variables:

  • kinematics (Kinematics) – Kinematic variables and deformation tensors.

  • bulk (OgdenCiarletGeymonatElasticMaterial) – The volumetric component.

  • dev (NeoHookeanMooneyRivlinElasticMaterial) – The deviatoric component.

  • Psi (UFL expression) – Total strain energy density.

  • Sigma (UFL expression) – Total Second Piola-Kirchhoff stress tensor.

  • p_hydro (UFL expression) – Hydrostatic pressure.

  • Sigma_dev (UFL expression) – Deviatoric part of the Second Piola-Kirchhoff stress.

  • Sigma_VM (UFL expression) – Von Mises equivalent stress calculated from \(\mathbf{\Sigma}_{dev}\).
