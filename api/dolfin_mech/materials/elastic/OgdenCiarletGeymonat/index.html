dolfin_mech.materials.elastic.OgdenCiarletGeymonat

Ciarlet-Geymonat volumetric elastic material implementation.

This module provides a strictly convex volumetric potential used to penalize volume changes and prevent numerical inversion in hyperelastic formulations.

class dolfin_mech.materials.elastic.OgdenCiarletGeymonat.OgdenCiarletGeymonat(kinematics, parameters, decoup=False)

Bases: dolfin_mech.materials.elastic.Elastic.ElasticMaterial

Class representing the Ciarlet-Geymonat volumetric elastic material model.

This model provides a strictly convex volumetric strain energy density functional. It is often used as the compressible part of a hyperelastic formulation to penalize volume changes and prevent element inversion.

The strain energy density \(\Psi\) is defined as:

\[\Psi = C_0 (J^2 - 1 - 2\ln(J))\]
Where:

  • \(J = \det(\mathbf{F})\) is the volume ratio (Jacobian).

  • \(C_0\) is a material parameter related to the bulk modulus.

The second Piola-Kirchhoff stress \(\mathbf{\Sigma}\) is:

\[\mathbf{\Sigma} = 2 C_0 (J^2 - 1) \mathbf{C}^{-1}\]
Variables:

  • kinematics (Kinematics) – Object containing kinematic variables (J, C_inv, etc.).

  • C0 (float) – Material constant scaling the volumetric response.

  • Psi (UFL expression) – Volumetric strain energy density.

  • Sigma (UFL expression) – Second Piola-Kirchhoff stress tensor.

  • P (UFL expression) – First Piola-Kirchhoff stress tensor.

  • sigma (UFL expression) – Cauchy stress tensor.

  • checkJ (bool) – If True, adds a conditional check to ensure \(J > 0\) during stress evaluation.
