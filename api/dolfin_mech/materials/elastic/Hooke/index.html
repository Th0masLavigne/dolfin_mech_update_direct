dolfin_mech.materials.elastic.Hooke

Implementation of linear elastic material models based on Hooke’s law.

This module provides classes for standard isotropic linear elasticity, as well as its decomposition into volumetric (bulk) and deviatoric (shear) components. These models are designed for use with linearized kinematics.

Module Contents

class dolfin_mech.materials.elastic.Hooke.Hooke(kinematics, parameters)

Bases: dolfin_mech.materials.elastic.Elastic.ElasticMaterial

Class implementing the classical isotropic linear elastic Hooke material model.

This model relates the Cauchy stress \(\boldsymbol{\sigma}\) to the infinitesimal strain \(\boldsymbol{\varepsilon}\) through the Lamé parameters \(\lambda\) and \(\mu\).

The strain energy density \(\psi\) is defined as:

\[\psi = \frac{\lambda}{2} (\text{tr } \boldsymbol{\varepsilon})^2 + \mu \boldsymbol{\varepsilon} : \boldsymbol{\varepsilon}\]
Variables:

  • kinematics (LinearizedKinematics) – Kinematic object providing the strain tensor \(\boldsymbol{\varepsilon}\).

  • lmbda (dolfin.Constant) – First Lamé parameter \(\lambda\).

  • mu (dolfin.Constant) – Second Lamé parameter (shear modulus) \(\mu\).

  • psi (ufl.Form) – Strain energy density function.

  • sigma (ufl.Form) – Cauchy stress tensor \(\boldsymbol{\sigma} = \lambda (\text{tr } boldsymbol{\varepsilon}) \mathbf{I} + 2\mu\boldsymbol{\varepsilon}\).

class dolfin_mech.materials.elastic.Hooke.HookeBulk(kinematics, parameters)

Bases: dolfin_mech.materials.elastic.Elastic.ElasticMaterial

Class implementing the spherical (bulk) part of the Hooke material model.

This model captures only the volumetric response, relating the pressure to the trace of the strain tensor.

The volumetric strain energy density is defined as:

\[\psi = \frac{d K}{2} (\text{tr } \boldsymbol{\varepsilon}_{sph})^2\]

where \(d\) is the spatial dimension and \(K\) is the bulk modulus.

class dolfin_mech.materials.elastic.Hooke.HookeDev(kinematics, parameters)

Bases: dolfin_mech.materials.elastic.Elastic.ElasticMaterial

Class implementing the deviatoric (shear) part of the Hooke material model.

This model captures the shape-changing response at constant volume, relating the deviatoric stress to the deviatoric strain.

The deviatoric strain energy density is defined as:

\[\psi = G \boldsymbol{\varepsilon}_{dev} : \boldsymbol{\varepsilon}_{dev}\]

where \(G\) is the shear modulus (equivalent to \(\mu\)).
