dolfin_mech.kinematics.Kinematics

Kinematics implementation for non-linear mechanics.

This module provides the Kinematics class which handles the computation and storage of deformation tensors, invariants, and frame transformations.

class dolfin_mech.kinematics.Kinematics.Kinematics(U, U_old=None, Q_expr=None)

Class to compute and store kinematic quantities for non-linear solid mechanics.

This class computes standard deformation tensors, their invariants, and isochoric (bar) versions based on the displacement field \(\mathbf{U}\). It also supports time-stepping (old states) and local frame transformations.

Parameters:

  • U (dolfin.Function) – Current displacement field.

  • U_old (dolfin.Function, optional) – Displacement field from the previous time step. Defaults to None.

  • Q_expr (dolfin.Expression, optional) – Rotation matrix for local coordinate systems. Defaults to None.

Variables:

  • U (dolfin.Function) – The current displacement field.

  • dim (int) – Spatial dimension.

  • I (dolfin.Identity) – Identity tensor of dimension dim.

  • F (ufl.Form) – Deformation gradient tensor \(\mathbf{F} = \mathbf{I} + \nabla \mathbf{U}\).

  • J (ufl.Form) – Determinant of the deformation gradient \(J = \det(\mathbf{F})\).

  • C (ufl.Form) – Right Cauchy-Green deformation tensor \(\mathbf{C} = \mathbf{F}^T \mathbf{F}\).

  • E (ufl.Form) – Green-Lagrange strain tensor \(\mathbf{E} = \mathbf{C} - \mathbf{I}\).

  • F_bar (ufl.Form) – Isochoric deformation gradient \(\bar{\mathbf{F}} = J^{-1/d} \mathbf{F}\).

  • C_bar (ufl.Form) – Isochoric Right Cauchy-Green tensor \(\bar{\mathbf{C}} = \bar{\mathbf{F}}^T \bar{\mathbf{F}}\).
