dolfin_mech.kinematics.inversekinematics

Inverse kinematics implementation for non-linear solid mechanics.

This module provides the InverseKinematics class, which computes deformation quantities based on a displacement field defined on the deformed configuration.

Module Contents

class dolfin_mech.kinematics.inversekinematics.InverseKinematics(u, u_old=None)

Class to compute and store kinematic quantities for inverse non-linear solid mechanics.

In the inverse formulation, the displacement field \(\mathbf{u}\) is defined on the deformed (spatial) configuration. The deformation gradient \(\mathbf{F}\) is computed as the inverse of the mapping from the reference to the deformed configuration: \(\mathbf{F} = (\mathbf{I} + \nabla \mathbf{u})^{-1}\).

Parameters:

  • u (dolfin.Function) – Current displacement field defined on the spatial mesh.

  • u_old (dolfin.Function, optional) – Displacement field from the previous time step. Defaults to None.

Variables:

  • u (dolfin.Function) – The current displacement field in the spatial configuration.

  • dim (int) – Spatial dimension.

  • I (dolfin.Identity) – Identity tensor of dimension dim.

  • f (ufl.Form) – Spatial displacement gradient \(\mathbf{f} = \mathbf{I} + \nabla \mathbf{u}\).

  • F (ufl.Form) – Deformation gradient tensor \(\mathbf{F} = \mathbf{f}^{-1}\).

  • J (ufl.Form) – Determinant of the deformation gradient \(J = \det(\mathbf{F})\).

  • C (ufl.Form) – Right Cauchy-Green deformation tensor \(\mathbf{C} = \mathbf{F}^T \mathbf{F}\).

  • E (ufl.Form) – Green-Lagrange strain tensor \(\mathbf{E} = \frac{1}{2}(\mathbf{C} - \mathbf{I})\).

  • F_bar (ufl.Form) – Isochoric deformation gradient \(\bar{\mathbf{F}} = J^{-1/d} \mathbf{F}\).

  • C_bar (ufl.Form) – Isochoric Right Cauchy-Green tensor \(\bar{\mathbf{C}} = \bar{\mathbf{F}}^T \bar{\mathbf{F}}\).
