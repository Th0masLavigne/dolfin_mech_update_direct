.. py:function:: compute_error(val, ref)

   Compute the relative error or ratio between a value and a reference.

   The error is calculated as:

   .. math::

       e = \begin{cases} 0 & \text{if } \text{ref} \approx 0 \text{ and } \text{val} \approx 0 \\ 1 & \text{if }
       \text{ref} \approx 0 \text{ and } \text{val} \neq 0 \\ \frac{\text{val}}{\text{ref}} & \text{otherwise}
       \end{cases}

   :Parameters: * **val** (:py:class:`float`) -- The current or calculated value to compare.
                * **ref** (:py:class:`float`) -- The reference or baseline value.

   :returns: The computed relative error or ratio.
   :rtype: float

   .. rubric:: Notes

   This function uses :py:func:`dolfin.near` with a tolerance of :math:`10^{-9}`
   to handle floating point comparisons near zero.
