.. py:class:: Kinematics(U, U_old=None, Q_expr=None)

   Class to compute and store kinematic quantities for non-linear solid mechanics.

   This class computes standard deformation tensors, their invariants, and
   isochoric (bar) versions based on the displacement field :math:`\mathbf{U}`.
   It also supports time-stepping (old states) and local frame transformations.

   :Parameters: * **U** (:py:class:`dolfin.Function`) -- Current displacement field.
                * **U_old** (:py:class:`dolfin.Function, optional`) -- Displacement field from the previous
                  time step. Defaults to None.
                * **Q_expr** (:py:class:`dolfin.Expression, optional`) -- Rotation matrix for local
                  coordinate systems. Defaults to None.

   :ivar U: The current displacement field.
   :vartype U: :py:class:`dolfin.Function`
   :ivar dim: Spatial dimension.
   :vartype dim: :py:class:`int`
   :ivar I: Identity tensor of dimension ``dim``.
   :vartype I: :py:class:`dolfin.Identity`
   :ivar F: Deformation gradient tensor :math:`\mathbf{F} = \mathbf{I} + \nabla \mathbf{U}`.
   :vartype F: :py:class:`ufl.Form`
   :ivar J: Determinant of the deformation gradient :math:`J = \det(\mathbf{F})`.
   :vartype J: :py:class:`ufl.Form`
   :ivar C: Right Cauchy-Green deformation tensor :math:`\mathbf{C} = \mathbf{F}^T \mathbf{F}`.
   :vartype C: :py:class:`ufl.Form`
   :ivar E: Green-Lagrange strain tensor :math:`\mathbf{E} = \mathbf{C} - \mathbf{I}`.
   :vartype E: :py:class:`ufl.Form`
   :ivar F_bar: Isochoric deformation gradient :math:`\bar{\mathbf{F}} = J^{-1/d} \mathbf{F}`.
   :vartype F_bar: :py:class:`ufl.Form`
   :ivar C_bar: Isochoric Right Cauchy-Green tensor :math:`\bar{\mathbf{C}} = \bar{\mathbf{F}}^T \bar{\mathbf{F}}`.

   :vartype C_bar: :py:class:`ufl.Form`
